# Merrin (WIP)
A basic system monitor for Linux systems

## Demo
![Merrin demonstration](assets/demo.png)

## Metrics
* CPU Utilization %
* CPU Temperature
* GPU Utilization
* GPU Temp
* RAM Utilization
* Uptime

## Installation
Run the installation script
```
./install.sh
```

## Usage
```
merrin
```
